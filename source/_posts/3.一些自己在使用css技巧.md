### 省略

```javascript
    //单行省略
    width: 100px;
    white-space: nowrap;
    text-overflow: ellipsis;
    overflow: hidden;

    //多行省略

    overflow: hidden;
    text-overflow: ellipsis;
```

### input 功能

 1. 弹出数字键盘
```javascript
    // 有"#" "*"符号输入 
    <input type="tel" />
    // 纯数字
```

